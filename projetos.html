    Esse foi o meu primeiro projeto de jogo em que eu tentei usar o famigerado 3D, como um primeiro projeto já era de se esperar que não seria nada muito bom, e nem otimizado. Com um tema simples e genérico de terror onde o jogador deve achar certa quantidade de itens para finalizar o jogo, nesse você deve achar três fusíveis para ligar a energia novamente.



    Esse foi um projeto em equipe de um jogo para uma GameJam, onde tinhamos aproximadamente um mês para fazer um jogo com o tema "Nem tudo é o que parece". Era um jogo estilo FPS baseado em um sistema de "Waves" (hordas), onde deveriamos acabar com as hordas de inimigos para conseguir fugir do local onde o jogador fica preso, e consequentemente finalizar o jogo.



    Esse é o meu projeto mais recente de jogo, onde eu tentei usar terror psicológico em um jogo 2D, o jogo tem dois finais diferentes onde um deles é simples enquanto o outro é mais complicado. Foi um projeto feito em uma semana e nçao saiu muito bem como era planejado, mas ficou bem polido.
